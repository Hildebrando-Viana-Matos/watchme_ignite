## About The Project
<p>Welcome to the watchme project. It is a system in which the user can watch some films on any topic. I did this project at Rocketseat Ignite Bootcamp and used the React "Fast Refresh".</p>

### Built With
I used some web technologies in order to train:mortar_board::
* ReactJs
* TypeScript
* JSON
* Yarn Server
* MirageJS

## Getting Started
### Prerequisites
You need the Yarn installed in your local machine
  ```sh
  npm install --global yarn
  ```
If you don1t have npm in your local machine, install now:
  ```sh
  npm install npm@latest -g
  ```
### Installation
1. Clone the repository
  ```sh
  git clone git@github.com:Hildebrando-Viana-Matos/watchme_ignite.git
  ```
2. Install the dependencies
  ```bash
  yarn
  ```
3. Run the development server:
  ```bash
  yarn start
  ```
4. And now, you can go to your browser to go to localhost
